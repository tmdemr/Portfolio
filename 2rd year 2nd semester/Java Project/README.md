# Java Project

## 개요
-  정보가 담긴 Open API를 활용한 병원의 공실, 전화번호, 위치, 병원 이름 등의 정보를 표시하는 프로그램

## 특징 목록
- OPNE API의 XML 데이터 파싱
- Swing을 활용한 GUI
- Google Map API를 활용한 위치 검색
- 소켓 통신을 활용한 채팅방 구현
- JTable을 활용한 사용자가 필요로 하는 데이터 표시
  - 사용자가 검색한 위치에 속한 병원의 목록을 표시
