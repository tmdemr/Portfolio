## iterator
- 리스트의 반복.
- 배열과는 달리 무한히 값을 넣을 수 있음.
- a.next, a.pop 등으로 리스트보다 더 간단하게 프로그래밍이 가능함.
- 예를 들어, 이벤트를 진행하는데 얼마나 사람들이 참석할 지 모르는 경우 등에 사용하면 편리할 수 있음.
- iter : 매개변수가 iterator인지 아닌지 알려주는 함수.
