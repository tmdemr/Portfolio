# Visual Programming Project

## 개요
-  음원을 버튼에 등록하면, 버튼에 매핑되는 키보드를 입력시 해당되는 음을 서버에 연결된 모든 클라이언트에 즉시 출력하는 합주 프로그램

## 특징 목록
- C++ 구현
- 인터넷이 연결되어 있다면, 키보드 입력 즉시 모든 클라이언트에 음 출력
- MFC를 활용한 GUI
- preTranslateMessage 패키지를 활용한 포커싱 오류 해결
