# Network Model

## OSI Model

### 계층 구조

  1. 물리층
  2. 데이터 링크
  3. 네트워크
  4. 전송층

  (사용자)
  5. 세션
  6. 표현
  7. 응용

## 대등 대 대등 프로세스(Peer to Peer)
- 해당 계층에서 통신하는 각 장치의 프로세스

1, 2, 3 계층은 네트워크 지원계층.  
한 장치에서 다른 장치로 데이터를 이동할 때 필요한 물리적인 면을 처리.

전송층은 종단 대 종단까지 신뢰할 수 있는 데이터 전송을 보장.

5, 6, 7 계층은 사용자 지원 계층
